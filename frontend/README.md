# Career Path AI Frontend

Frontend React application for the vocational questionnaire system.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:5173` in your browser.

## Features

- Home page with project introduction
- Questionnaire page that fetches questions from the backend
- Results page displaying AI-generated career recommendations
- Responsive design using Material-UI
- Navigation with React Router

## Backend Integration

Ensure the backend is running on `http://localhost:3000` for API calls.

## Build

To build for production:
```bash
npm run build
```
